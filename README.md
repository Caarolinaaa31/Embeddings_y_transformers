# Embeddings y transformers
Este repositorio incluye el trabajo final de máster titulado: "Impacto de la generación de Embeddings en tareas con Arquitectura Transformer". La tarea analizada es un clasificador multietiqueta de resúmenes de artículos matemáticos. Se incluye el código (en .ipnyb & .py), una captura del código en .pdf y la memoria del trabajo.

Para la ejecución del código se recomienda descargar el .ipnyb o el .pdf del código y leer las instrucciones. Así podreis probar el código de una manera más rápida.
